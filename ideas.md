# Ideias de Design - Meu Direito Previdenciário

## Abordagem Escolhida: Modernismo Institucional com Humanidade

### Design Movement
**Modernismo Institucional Humanizado** - Combina a clareza e confiabilidade de design institucional com elementos visuais que transmitem empatia, acessibilidade e inclusão. Inspirado em órgãos públicos contemporâneos que buscam aproximação com o cidadão.

### Core Principles
1. **Clareza Radical**: Informações complexas apresentadas de forma progressiva e intuitiva, sem jargão desnecessário
2. **Confiança Visual**: Paleta profissional mas acessível, que comunica segurança sem frieza
3. **Humanidade Inclusiva**: Imagens reais de pessoas, tipografia legível, espaçamento generoso
4. **Hierarquia Funcional**: Guia visual claro que prioriza o que importa (benefícios, como pedir)

### Color Philosophy
- **Primária**: Azul profundo (`#0a4d8c` → `#1a6fb0`) - transmite confiança, segurança, institucionalidade
- **Secundária**: Laranja vibrante (`#ff8c42`) - humanidade, ação, otimismo
- **Neutros**: Branco limpo, cinzas suaves - legibilidade máxima
- **Fundos**: Azul muito claro (`#f0f7ff`) - acolhedor sem ser infantil

### Layout Paradigm
- **Hero assimétrico**: Imagem contextualizada à direita, texto à esquerda (mobile: empilhado)
- **Cards com propósito**: Não apenas caixas, mas seções com visual que indica função (alerta, dica, informação)
- **Navegação sticky**: Menu fixo no topo para acesso rápido às seções principais
- **Seções com respiro**: Uso generoso de whitespace entre cards, não comprimido

### Signature Elements
1. **Barra esquerda colorida em cards**: Barra laranja/azul à esquerda de cada seção principal (já presente no original)
2. **Ícones contextuais**: Emojis mantidos estrategicamente (não excessivos) para humanizar
3. **Tabelas com ritmo visual**: Linhas alternadas, cabeçalho azul escuro, tipografia clara

### Interaction Philosophy
- Transições suaves ao passar hover em cards (elevação sutil)
- Botão "voltar ao topo" fluido, não invasivo
- Links com underline ao hover, mantendo acessibilidade
- Navegação suave entre seções (scroll behavior smooth)

### Animation
- **Entrada de cards**: Fade-in suave ao scroll (opcional, não obrigatório)
- **Hover em cards**: translateY(-4px) com sombra aumentada
- **Botões**: Mudança de cor suave (0.3s), sem pular
- **Links de navegação**: Fundo suave ao hover, sem flash

### Typography System
- **Display/Títulos**: Segoe UI Bold, 1.6-2rem, cor azul profundo
- **Subtítulos**: Segoe UI SemiBold, 1.1-1.3rem, cor azul médio
- **Body**: Segoe UI Regular, 1rem, cor escura, line-height 1.6
- **Pequeno (labels, tabelas)**: 0.9rem, cor cinza
- **Hierarquia**: Peso e tamanho criam distinção clara, sem muitas fontes
